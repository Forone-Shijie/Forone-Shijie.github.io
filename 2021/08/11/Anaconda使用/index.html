Anaconda基本使用

最近由于在参与实验室 高速视觉感知课题的研究，通过不断地了解他人的研究成果，我们最终暂时地将重点转移到了事件相机与RGB相机的数据融合部分中。关于这些方面的细节呢，后面有相关进展了我会继续写相应的博客，由于用到了Python去处理我们的数据，我之前有简单地学习过Python的使用，但是非科班出身，也没有太多的使用经历，对于一种编程语言的掌握终究还是差些意思。

这里整理了一下基本使用，方便忘记了之后来复习。

设置国内镜像

Anaconda的服务器在国外，安装多个packages时，conda下载的速度经常很慢。清华TUNA镜像源有Anaconda仓库的镜像，将其加入conda的配置即可：

  • 添加Anaconda的TUNA镜像

    conda config --add channels https://mirrors.tuna.tsinghua.edu.cn

  • TUNA的help中镜像地址加有引号，需要去掉

  • 设置搜索时显示通道地址

    conda config --set show_channel_urls yes

  • 恢复默认镜像

    conda config --remove-key channels
